# App Mata Mosquito

Um jogo simples feito com **HTML, CSS e JavaScript** onde o objetivo é clicar nos mosquitos antes que eles escapem. É um projeto divertido para aprender lógica básica de manipulação de DOM, eventos de clique e controle de telas de jogo.

## Sobre

Esse projeto é um jogo de navegador no estilo *clicker*. O jogador precisa clicar no mosquito que aparece na tela para marcar pontos antes que o tempo acabe.

## Tecnologias

O jogo foi desenvolvido com:

- **HTML5**
- **CSS3**
- **JavaScript (vanilla)**

## Como Jogar

1. Abra o arquivo `index.html` no navegador.  
2. O jogo começa automaticamente.  
3. Clique nos mosquitos que aparecem antes que somem.  
4. Se você deixar o mosquito escapar, perde vida.  
5. Quando as vidas acabam, o jogo termina.

Os principais arquivos:

- `index.html` – página principal do jogo  
- `jogo.js` – lógica do jogo  
- `estilo.css` – estilos visuais  
- `app.html`, `fim_de_jogo.html`, `vitoria.html` – telas do jogo  
- Imagens (.png/.jpg) para elementos visuais
